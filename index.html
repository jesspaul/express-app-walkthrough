Express App Walkthrough

A step by step look at creating a simple CRUD todo app with Express.

  1. Set up file structure

    1. Create project folder

      In terminal:

      mkdir express-todo-app

    2. Create server.js file

      In terminal in project folder:

      touch server.js

    3. Initialize npm with defaults

      In terminal:

      npm init -y

      This creates package.json

    4. Install npm packages

      In terminal:

      npm i express ejs morgan method-override

    5. Setup MVC folder structure

      In terminal:

      mkdir models views controllers routes

  2. Set up the server

    1. Open folder in VS Code

      In terminal:

      code .

    2. Load in express

      In server.js:

      const express = require('express');
      const app = express();

    3. Set the view engine

      In server.js:

      app.set('view engine', 'ejs');

    4. Mount middleware

      In server.js:

      app.use(morgan('dev'));
      app.use(express.static('public'));
      app.use(express.urlencoded({ extended: false }));
      app.use(methodOverride('_method'));

    5. Listen on a port

      In server.js:

      app.listen(3000, function() {
        console.log('Express is listening on port 3000');
      });

    6. Start the app

      In terminal:

      nodemon

    7. View the app

      In the browser:

      localhost:3000
