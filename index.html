Express App Walkthrough

A step by step look at creating a simple CRUD todo app with Express.

  1. Set up file structure

    1. Create project folder

      In terminal:

      mkdir express-todo-app

    2. Create server.js file

      In terminal in project folder:

      touch server.js

    3. Initialize npm with defaults

      In terminal:

      npm init -y

      *Note: This creates package.json

    4. Install npm packages

      In terminal:

      npm i express ejs morgan method-override

    5. Setup MVC folder structure

      In terminal:

      mkdir models views controllers routes

  2. Set up the server

    1. Open folder in VS Code

      In terminal:

      code .

    2. Load in express

      In server.js:

      const express = require('express');
      const app = express();

    3. Load in middleware

      In server.js:

      const morgan = require('morgan');
      const methodOverride = require('method-override');

    4. Set the view engine

      In server.js:

      app.set('view engine', 'ejs');

    5. Mount middleware

      In server.js:

      app.use(morgan('dev'));
      app.use(express.static('public'));
      app.use(express.urlencoded({ extended: false }));
      app.use(methodOverride('_method'));

    6. Listen on a port

      In server.js:

      app.listen(3000, function() {
      console.log('Express is listening on port 3000');
      });

    7. Start the app

      In terminal:

      nodemon

    8. View the app

      In the browser:

      localhost:3000

      *Note: The page will return a 404 error at this point.

  3. Set up the "database"

    1. Create a file to act as the "database"

      In terminal:

      touch models/todo.js

      *Note: The file is written as singular by convention

    2. Create an array of todo objects

      In models/todo.js:

      const todos = [
      {text: 'Feed Dogs', done: true},
      {text: 'Learn Express', done: false},
      {text: 'Buy Milk', done: false}
      ];

    3. Create methods to access one or all todo objects

      In models/todo.js:

      function getAll() {
      return todos;
      };
      function getOne(id) {
      return todos[id];
      };

    4. Export the methods

      In models/todo.js:

      module.exports = {
      getAll,
      getOne,
      };

  4. Set up home page

    1. Create index view and route file

      In terminal:

      touch views/index.ejs routes/index.js

    2. Create home page

      In index.ejs:

      ! for boilerplate HTML

      Write HTML content in the body

    3. Load Express in index router

      In routes/index.js:

      const express = require('express');
      const router = express.Router();

    4. Create index router

      In routes/index.js:

      router.get('/', function(req, res) {
      res.render('index');
      });

      *Note: res.render takes in a string of the ejs file name inside the views folder

    5. Export index router

      In routes/index.js:

      module.exports = router;

    6. Load index router in server

      In server.js:

      const indexRouter = require('./routes/index');

      *Note: File path must start with ./ for a local file

    7. Mount index route

      In server.js:

      app.use('/', indexRouter);

      *Note: localhost:3000 will now display the home page

  5. Set up todos index

    1. Create folder and index file for todos view

      In terminal:

      mkdir views/todos
      touch views/todos/index.ejs

    2. Write basic content for todos index

      In views/todos/index.ejs:

      ! for boilerplate HTML

      Write basic HTML content in the body

    3. Link to the todos index from the home page

      In views/index.ejs:

      <a href="/todos">Todo List</a>

    4. Create todos route file

      In terminal:

      touch routes/todos.js

    5. Load Express in todos router

      In routes/todos.js:

      const express = require('express');
      const router = express.Router();

    6. Create todos get index route

      In routes/todos.js:

      router.get('/', todosCtrl.index);

    7. Create todos controller file

      In terminal:

      touch controllers/todos.js

    8. Load model data in the controller

      In controllers/todos.js:

      const Todo = require('../models/todo');

    9. Write function to render todos index page with model data

      In controllers/todos.js:

      function index(req, res) {
      res.render('todos/index', {
      todos: Todo.getAll(),
      });
      };

    10. Export index function

      In controllers/todos.js:

      module.exports = {
      index
      };

    11. Load controller function in the router

      In routes/todos.js:

      const express = require('../controllers/todos');

    12. Export todos router

      In routes/todos.js:

      module.exports = router;

    13. Load todos router in server

      In server.js:

      const todosRouter = require('./routes/todos');

    14. Mount todos route

      In server.js:

      app.use('/todos', todosRouter);

      *Note: localhost:3000/todos will now display the todos index view

    15. Programatically create list of todos data in index view

      In views/todos/index.ejs:

      <ul>
      <% todos.forEach(function(todo, idx) { %>
      <li>
      <%= todo.text %> : <%= todo.done ? 'Done' : 'Not done' %>
      </li>
      <% }); %>
      </ul>

  6. Set up the show functionality

    1. Add link to each todo item's id in the index view

      In views/todos/index.ejs:

      <a href="/todos/<%= idx %>"><%= todo.text %></a>

    2. Add a show route

      In routes/todos.js:

      router.get('/:id', todosCtrl.show);

    3. Create the show controller function with model data

      In controllers/todos.js:

      function show(req, res) {
      res.render('todos/show', {
      todo: Todo.getOne(req.params.id),
      todoNum: parseInt(req.params.id) + 1
      });
      };

    4. Export the show function

      In controllers/todos.js:

      module.exports = {
      index,
      show
      };

    5. Create the show view

      In terminal:

      touch views/todos/show.ejs

    6. Write content for show view

      In views/todos/show.ejs:

      ! for boilerplate HTML and add the following into the body:

      <a href="/">Home</a>
      <a href="/todos">All Todos</a>
      <h1>Todo #<%= todoNum %></h1>
      <h3><%= todo.text %></h3>
      <h3>Complete: <%= todo.done ? 'Yes' : 'No' %></h3>

      *Note: localhost:3000/todos/:id will now display the todos show view

  7. Add new todo

    1. Add a link in the todos index to create a new todo

      In views/todos/index.ejs:

      <a href="/todos/new">Add Todo</a>

    2. Add the route for a page to add a new todo

      In routes/todos.js:

      router.get('/new', todosCtrl.new);

      *Note: This must be written above the get /:id route so /new is not read as an id

    3. Add the controller function for a new todo

      In controllers/todos.js:

      function newTodo(req, res) {
      res.render('todos/new');
      };

      *Note: The user is creating the new todo, so we do not need to pass in any data to the page

    4. Export the new controller function

      In controllers/todos.js:

      module.exports = {
      ...
      new: newTodo
      };

    5. Create the new view

      In terminal:

      touch views/todos/new.ejs

    6. Write content for new view

      In views/todos/new.ejs:

      ! for boilerplate HTML and add the following into the body:

      <h1>New Todo</h1>
      <form action="/todos" method="POST">
      <input type="text" name="text">
      <button type="submit">Save Todo</button>
      </form>

    7. Add the create route

      In routes/todos.js:

      router.post('/', todosCtrl.create);

    8. Create the create controller function

      In controllers/todos.js:

      function create(req, res) {
      req.body.done = false;
      Todo.create(req.body);
      res.redirect('/todos');
      };

      *Note: Initialize the done property to false

      Add the new todo to the database - this method has not been created yet

      Redirect the user to the todos index

    9. Export the create controller function

      In controllers/todos.js:

      module.exports = {
      ...
      create
      };

    10. Add the create method to the model

      In models/todo.js:

      function create(todo) {
      todos.push(todo);
      };

    11. Export the create method

      In models/todo.js:

      module.exports = {
      ...
      create
      };

  8. Delete a todo

    1. Create a form to use as a delete button before each todo in the index

      In views/todos/index.ejs:

      <form action="/todos/<%= idx %>?_method=DELETE" method="post"></form>
      <button type="submit">X</button>
      </form>

      *Note: Use method-override to change form method to delete

      Optionally add CSS to display the form inline with the todo item

    2. Add the delete route

      In routes/todos.js:

      router.delete('/:id', todosCtrl.delete);

    3. Create the delete controller function

      In controllers/todos.js:

      function deleteTodo(req, res) {
      Todo.deleteOne(req.params.id);
      res.redirect('/todos');
      };

    4. Export the delete controller function

      In controllers/todos.js:

      module.exports = {
      ...
      delete: deleteTodo
      };

    5. Add the delete method to the model

      In models/todo.js:

      function deleteOne(todo) {
      todos.splice(id, 1);
      };

    6. Export the delete method

      In models/todo.js:

      module.exports = {
      ...
      deleteOne
      };

  9. Edit a todo

    1. Add a link to edit todo in show view

      In views/todos/show.ejs:

      <a href="/todos/<%= todoNum - 1 %>/edit">Edit This Todo</a>

    2. Add the route for a page to edit the todo

      In routes/todos.js:

      router.get('/:id/edit', todosCtrl.edit);

    3. Create the edit controller function

      In controllers/todos.js:

      function edit(req, res) {
      const todo = Todo.getOne(req.params.id);
      res.render('todos/edit', {
      todo,
      todoId: req.params.id,
      };

    4. Export the edit controller function

      In controllers/todos.js:

      module.exports = {
      ...
      edit
      };

    5. Create the edit view

      In terminal:

      touch views/todos/edit.ejs:

    6. Write content for edit view

      In views/todos/edit.ejs:

      ! for boilerplate HTML and add the following into the body:

      <h1>Edit Todo: <%= todo.text %></h1>
      <form action="/todos/<%= todoId %>/?_method=PUT" method="post"></form>
      <label>Description:
      <input type="text" name="text" value="<%= todo.text %>">
      </label>
      <label>Done:
      <input type="checkbox" name="done" <%- todo.done ? 'checked' : ''%>>
      </label>
      <button type="submit">Edit Todo</button>
      </form>

      *Note: Use method-override to change form method to put

      Use value and checked input attributes to prefill information

    7. Add the put route

      In routes/todos.js:

      router.put('/:id', todosCtrl.update);

    8. Create the update controller function

      In controllers/todos.js:

      function update(req, res) {
      req.body.done = req.body.done ? true : false;
      Todo.updateOne(req.params.id, req.body);
      res.redirect(`/todos/${req.params.id}`);
      };

      *Note: Set done property based on if checkbox is checked or not

      Update the todo with it's id and new text - this method hasn't been created yet

      Redirect the user to the show view

    9. Export the update controller function

      In controllers/todos.js:

      module.exports = {
      ...
      update
      };

    10. Add the update method to the model

      In models/todo.js:

      function updateOne(id, updatedTodo) {
      todos.splice(id, 1, updatedTodo);
      };

    11. Export the update method

      In models/todo.js:

      module.exports = {
      ...
      updateOne
      };
